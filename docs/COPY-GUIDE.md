## 🚀 Hero Section

### Tag
```
חדש בישראל - לא עוד בוט. סוכן.
```
> **למה?** מבדיל אותנו מצ'אטבוטים. "סוכן" = פעיל, חכם, עצמאי.

### Headline (H1)
```
צוות שלם שעובד בשבילך, [בלי לבקש משכורת]
```
- חלק מודגש = עם shimmer
- 4-6 מילים עיקריות = שיווק קלאסי

> **למה?** משלב 2 רעיונות:
> 1. "צוות שלם" - הבטחת ערך גדולה
> 2. "בלי לבקש משכורת" - חד, מצחיק, ייחודי. לא קלישאת AI.

### Subheadline
```
סוכני Spike עושים את העבודה שפעם היית משלם עליה אלפים: 
מניהול הרשתות ועד סינון לידים ובקרת איכות. 
כל העסק שלך מתופעל 24/7, בלי הוצאות שכר ובלי כאבי ראש.
```

> **דקויות:**
> - "פעם היית משלם" = עיגון בעבר אמיתי, לא הבטחה ערטילאית
> - "אלפים" = מציאותי (לא "מאות אלפים" - לא אמין)
> - "מניהול הרשתות ועד סינון לידים" = פירוט מוחשי
> - "24/7" = הבטחה משכנעת אבל לא מוגזמת

### Trust Line
```
✓ הקמה תוך 7 ימים  ✓ בלי התחייבות  ✓ עברית מלאה
```
> **למה אלה ה-3?**
> - **7 ימים** = משכנע (לא חודשים)
> - **בלי התחייבות** = מסיר חרדה ראשית
> - **עברית מלאה** = מבדיל ממוצרים זרים

### Hero Form
**שדות:**
- שם מלא
- טלפון
- אימייל

**כפתור:**
```
🚀 קבל הצעה אישית
```

**תת-משפט:**
```
⚡ נחזור אליך תוך 24 שעות. בלי התחייבות.
```

> **למה רק 3 שדות?** כל שדה נוסף = 10% פחות המרה. תחום העסק - נשאל בשיחה.

### 6 השאלות (סדר חשוב!)

#### 1. כמה זה עולה? (פתוחה כברירת מחדל)
```
כל עסק שונה - לכן אנחנו לא מציגים מחירים גנריים שלא תואמים לך. 
בשיחה איתנו נבין מה אתה צריך, ונבנה הצעה אישית מותאמת לעסק 
ולתקציב שלך. אנחנו פתוחים לעבוד עם עסקים בכל הגדלים.
```

#### 2. האם הנתונים שלי בטוחים?
```
לחלוטין. הסוכנים שלנו פועלים עם הרשאות מוגבלות ומאובטחות בלבד - 
הם רואים רק את מה שאתה מאפשר. כל הנתונים נשארים בחשבונות שלך 
(גוגל, טלגרם וכו'), ואנחנו לא שומרים מידע אצלנו. 
אתה תמיד יכול לבטל גישה בלחיצה אחת.
```

#### 3. אני לא מבין במחשבים. זה יסתבך לי?
```
לא. כל ההתקנה וההגדרות נעשות על ידינו - אתה לא צריך לעשות שום 
דבר טכני. אחרי ההקמה, כל מה שאתה צריך זה לפתוח את טלגרם ולקרוא 
את הדוחות. אם משהו לא ברור - אנחנו תמיד זמינים בעברית.
```

#### 4. מתי אראה תוצאות?
```
תוך 7 ימים מהאישור הסוכנים כבר עובדים ושולחים לך דוחות. 
תוך שבועיים-שלושה כבר תרגיש בהבדל - יותר זמן ביד, פחות לידים 
שנופלים בין הכיסאות, ותחושה ברורה של שליטה בעסק. 
אנחנו רואים תוצאות אצל הלקוחות שלנו תוך החודש הראשון.
```

#### 5. מה אם אני לא מרוצה?
```
בלי התחייבות לתקופה - אתה יכול לעצור מתי שתרצה, בלי קנסות 
ובלי הסברים. בנוסף, אנחנו עובדים איתך תוך כדי תנועה - 
אם משהו לא עובד מספיק טוב, אנחנו מכוונים את הסוכן. 
המטרה שלנו היא שתישאר כי אתה רוצה, לא כי אתה חייב.
```

#### 6. האם זה מתאים לעסק קטן?
```
דווקא לעסקים קטנים זה הכי משתלם. בעל עסק קטן הוא בדרך כלל גם 
המנהל, גם המכירות, גם השיווק וגם השירות - הסוכנים שלנו מורידים 
לך משם המון עומס. אנחנו עובדים עם עסקים של אדם אחד עד עסקים 
עם 20 עובדים.
```

## 🎯 עקרונות הקופי

### ✅ עושים
- 🇮🇱 עברית טבעית, ישראלית, חמה
- 👤 פנייה ב"אתה" (לא "אתם")
- 🎯 ישיר וחד, בלי מילים מיותרות
- 💛 חיבור רגשי (שקט נפשי, שליטה)
- 📊 מספרים ספציפיים (7 ימים, 24 שעות, 5 דקות)
- 🎪 רגע מצחיק / חד פעם בשניים-שלושה משפטים

### ❌ לא עושים
- 🚫 קלישאות שיווקיות ("הופכים את האופן")
- 🚫 ז'רגון טכני יתר ("AI", "אוטומציה" - מועט)
- 🚫 תרגום מאנגלית ("רעיון 1 הוא...")
- 🚫 מספרי סוכנים ("5 סוכנים" - אולי יוסיף בעתיד)
