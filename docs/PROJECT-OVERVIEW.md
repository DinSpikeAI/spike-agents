# 📘 Spike AI Agents - Project Overview

> סקירה כללית של הפרויקט וקהל היעד

## 🎯 מה זה Spike AI Agents?

**Spike AI Agents** הוא שירות **Done-For-You** (לא SaaS) שמספק חבילת סוכני AI לעסקים ישראליים.

### מה זה אומר בפועל?

* 🤖 **9 סוכני AI** עובדים 24/7 ברקע של העסק
* 📊 הסוכנים סורקים את הדאטה של העסק (Google Business, לידים, רשתות חברתיות, ביקורות)
* 📱 שולחים **דוחות לטלגרם** של בעל העסק
* 🇮🇱 **בעברית מלאה**
* ⚙️ **אנחנו מקימים, מתחזקים, ומתאימים** - הלקוח רק נהנה

### מה זה **לא**?

* ❌ **לא צ'אטבוט** שעונה ללקוחות
* ❌ **לא SaaS** שצריך להתפעל לבד
* ❌ **לא אוטומציה גנרית** - הכל מותאם לעסק

> ⚠️ **כלל ברזל בשיווק:** המילה "בוט" אסורה. רק "סוכן" / "סוכני AI".

## 💼 המודל

### זרימת מכירה

```
1. בעל עסק רואה דף נחיתה (spikeai.co.il)
2. ממלא טופס (3-4 שדות)
3. אנחנו מקבלים מייל → חוזרים אליו תוך 24 שעות
4. שיחה קצרה לאבחון העסק
5. הצעה אישית מותאמת
6. אישור → 7 ימים → סוכן עובד
```

### עקרונות

* 🤐 **אין מחירים פומביים** – הצעה נבנית בשיחה
* 🔓 **בלי התחייבות לתקופה** - אפשר לעצור מתי שרוצים
* 🌱 **Bootstrap mode** – מינימום הוצאות, מקסימום ערך

### מה מבדל אותנו?

| אנחנו | מתחרים |
| --- | --- |
| Done-For-You מלא | "תתפעל לבד" |
| תמיכה בעברית | אנגלית |
| הקמה תוך 7 ימים | "שבועות-חודשים" |
| בלי שיחות מכירה אגרסיביות | לחץ של מכירה |
| התאמה אישית מלאה | פתרון גנרי |

## 🛡️ עקרונות יסוד

1. **אמינות מעל הכל** - לא להבטיח דברים שלא נספק
