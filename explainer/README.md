# 🎬 Spike AI — Explainer Video

סרטון הסבר אנימטיבי באורך 60 שניות, עם timeline scrubber, play/pause, וכתוביות בעברית RTL.

## 📁 קבצים בתיקייה

| קובץ | תפקיד |
|---|---|
| **`Spike Explainer Video.html`** | הקובץ הראשי — פתח אותו בדפדפן |
| `spike.jsx` | רכיב המסקוט (SVG וקטורי, בעל animatable parts) |
| `scenes-1-2.jsx` | סצנה 1 (Hook) + סצנה 2 (Meet the Team) |
| `scene-3.jsx` | סצנה 3 (Demo — 4 רבעונים) |
| `scenes-4-5.jsx` | סצנה 4 (Result) + סצנה 5 (CTA) |
| `../animations.jsx` | מנוע ה-Stage/Timeline (חיצוני, נטען מהפרויקט) |
| `spike-mascot-reference.png` | התמונה המקורית של ה-mascot מהאתר (להשוואה) |

## ▶️ איך לצפות

1. פתח את `Spike Explainer Video.html` בדפדפן (Chrome / Safari / Firefox)
2. הסרטון מתחיל אוטומטית
3. **קיצורים:**
   - `Space` — play/pause
   - `←` / `→` — הקפצה של 0.1 שניות (Shift = שנייה שלמה)
   - `0` או `Home` — חזרה להתחלה
   - גרירת ה-scrubber התחתון — קפיצה לכל נקודה

## 🎥 איך להוציא MP4

הקובץ הוא HTML+SVG עם timeline — לא ייצוא ישיר ל-וידאו. כדי להפיק MP4:

### אופציה 1 — Screen Recording (הכי פשוט)
- **Mac:** QuickTime → File → New Screen Recording → בחר את חלון הדפדפן → לחץ Play בסרטון
- **Windows:** Win+G (Game Bar) → Record
- **Cross-platform:** [OBS Studio](https://obsproject.com/) — חינמי, איכות גבוהה

**טיפ:** פתח את הסרטון ב-fullscreen (F11), שים את ה-scrubber על 0, התחל הקלטה, לחץ Space.

### אופציה 2 — תוסף לדפדפן
- "Awesome Screenshot & Screen Recorder" (Chrome)
- "Loom" (web-based)

## 🎙️ איך להוסיף קריינות

הסרטון לא מכיל קול — רק כתוביות מסונכרנות בעברית בתחתית המסך, עם timestamp markers (למשל `[0:09 – 0:20]`) שמסייעים לסנכרן voice-over.

**זרימה מומלצת:**
1. הקלט screen recording של הסרטון (אופציה 1 למעלה) → קובץ MP4 שקט
2. הקלט קריינות בעברית (Audacity / GarageBand / [ElevenLabs](https://elevenlabs.io/) עם קול עברי)
3. שלב את הוידאו והאודיו ב:
   - **CapCut** (חינמי, פשוט)
   - **DaVinci Resolve** (חינמי, מקצועי)
   - **Premiere Pro** / **Final Cut Pro**
4. עקוב אחרי ה-timestamps של הכתוביות לסנכרון

## ⚙️ עריכה

הקבצים JSX קריאים — אפשר לערוך:

- **טקסט:** חפש בקובץ הסצנה הרלוונטי
- **צבעים:** ערוך את ה-`hue` prop על ה-`SpikeMascot`, או את ה-CSS variables
- **תזמון:** בקובץ `Spike Explainer Video.html`, ה-`<Sprite start={...} end={...}>` קובע מתי כל סצנה רצה
- **Mascot:** `spike.jsx` — כל הגוף וקטורי וניתן לערוך (עיניים, ידיים, אנטנה...)
