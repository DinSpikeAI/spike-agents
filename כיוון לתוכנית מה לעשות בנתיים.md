## 📌 1. תקציר ביצועי

**מה:** חבילה של 9 סוכני AI שעובדים 24/7 ברקע של עסק ישראלי, סורקים דאטה ממקורות חיצוניים ופנימיים, ושולחים דוחות יומיים בטלגרם.

**למי:** עסקי שירות עירוניים (קליניקות, מספרות, מאמני כושר) + עסקי מכירות (סוכנים, חנויות, נדל"ן). הקהל מאופיין ב"נוכחות דיגיטלית בסיסית" — Google Business / אינסטגרם / וואטסאפ.

**מודל:** Done-For-You. הלקוח מבקש דמו → onboarding 30-60 דקות → סוכן רץ אצלך בענן ושולח לו לטלגרם.

**מחיר:** ₪1,800-6,000 הקמה + ₪500-1,800/חודש (6 חבילות, 2 קהלים).

**עלות תפעולית:** $5-15/חודש בהתחלה (Claude tokens בלבד), $76-136/חודש עם 3+ לקוחות.

**יעד 6 חודשים:** 5-8 לקוחות פעילים = ₪7,500-15,000 הכנסה חודשית, רווח גולמי ~85%.

**יתרון תחרותי:** המייסד כבר בנה מערכת זהה אדריכלית (Spike AI Studio — 13 סוכנים שרצים ב-Vercel + Supabase + Telegram + Claude). זה אדריכלות מוכחת, רק עם החלפה של מקורות, prompts, ו-tenant.

## 📌 5. מודל עסקי ותמחור

### 5.1 מודל
**Done-For-You (DFY) — לא SaaS.**
- הלקוח לא נרשם לבד
- הלקוח לא מגדיר לבד
- אנחנו עושים onboarding אישי (30-60 דקות)
- הסוכן רץ אצלנו בענן (לא אצל הלקוח)
- הלקוח מקבל דוחות לטלגרם הפרטי שלו

### 5.2 חבילות תמחור

#### לעסקי שירות (קטגוריה A)

| חבילה | הקמה | חודשי | סוכנים כלולים |
|---|---|---|---|
| **Basic** | ₪1,800 | ₪500 | בוקר + ניקיון |
| **Pro ⭐** | ₪3,200 | ₪900 | + רשתות + מעקב + ביקורות |
| **Full** | ₪4,500 | ₪1,400 | + מנהל + התאמה מלאה |

#### לעסקי מכירות (קטגוריה B)

| חבילה | הקמה | חודשי | סוכנים כלולים |
|---|---|---|---|
| **Sales Starter** | ₪2,500 | ₪700 | בוקר + לידים חמים |
| **Sales Pro ⭐** | ₪4,000 | ₪1,200 | + מכירות + מעקב + רשתות |
| **Sales Full** | ₪6,000 | ₪1,800 | + מנהל + מלאי + ביקורות |

### 5.3 הצדקת המחיר
**ROI שאתה מציג ללקוח:**
- 5 שעות/חודש של ניטור ידני × ₪200-400/שעה = **₪1,000-2,000/חודש בזמן שחסכת**
- אם הסוכן עוזר לסגור deal אחד נוסף בחודש = **₪5,000-50,000** (עסקי מכירות)
- אם הסוכן מונע 2 ביקורות שליליות שלא טופלו = **שווה ערך לחודש שלם של פרסום**

**עלות לקוח:** ₪500-1,800/חודש = **5x-25x ROI גלוי**.

### 5.4 חישוב יעד 6 חודשים
- **3 לקוחות Pro (קטגוריה A):** ₪900 × 3 = ₪2,700/חודש
- **2 לקוחות Sales Pro (קטגוריה B):** ₪1,200 × 2 = ₪2,400/חודש
- **סה"כ MRR:** ₪5,100/חודש
- **+ 2-3 setup חדשים בחודש:** ₪3,000-9,000 חד-פעמי
- **הכנסה חודשית כוללת:** ₪8,000-14,000

### 5.5 תקרת הצמיחה
מעל **₪10,000/חודש** ברוטו → צריך לעבור לעוסק מורשה (תקרת עוסק פטור 2026 ≈ ₪122,833/שנה). מצב "עסק זעיר" החדש (2024+) נותן ניכוי שטוח 30% וחוסך דוח שנתי.

**אסטרטגיה:** להישאר מתחת לתקרה במשך 6-9 חודשים. אחרי זה לעבור לעוסק מורשה.

### 6.3 Database Schema (Supabase Multi-Tenant)

```sql
-- טבלת לקוחות (הליבה של multi-tenant)
create table clients (
  id uuid primary key default gen_random_uuid(),
  name text not null,                    -- "סלון רקפת"
  slug text unique not null,             -- "rakefet-salon"
  
  -- Telegram (פר לקוח)
  telegram_bot_token text,               -- בוט פרטי שלהם
  telegram_chat_id text,                 -- ה-chat ID שלהם
  
  -- הקשר עסקי (JSONB גמיש)
  business_context jsonb,                -- {category, target, services, competitors, ...}
  
  -- אילו סוכנים פעילים
  active_agents text[],                  -- ['morning', 'social', 'manager']
  
  -- הגדרות פר סוכן
  agent_configs jsonb,                   -- {social: {tone: "warm", schedule: ["09:00"]}}
  
  -- אינטגרציות (OAuth tokens)
  google_sheets_id text,
  google_business_account_id text,
  calendly_token text,
  
  -- חבילה
  plan text,                             -- 'basic' / 'pro' / 'full' / 'sales_starter' / etc.
  status text default 'active',          -- 'active' / 'paused' / 'churned'
  
  created_at timestamptz default now(),
  updated_at timestamptz default now()
);

-- לידים (multi-tenant)
create table leads (
  id uuid primary key default gen_random_uuid(),
  client_id uuid references clients(id),
  source text,
  name text,
  email text,
  phone text,
  data jsonb,                            -- כל הפרטים נוספים
  score int,
  status text default 'new',             -- new, contacted, replied, hot, dead
  created_at timestamptz default now(),
  last_contact_at timestamptz
);

-- היסטוריית דוחות (לעקוב אחרי ביצועי הסוכן)
create table agent_runs (
  id uuid primary key default gen_random_uuid(),
  client_id uuid references clients(id),
  agent_name text,
  status text,                           -- success / failed
  output jsonb,
  error text,
  duration_ms int,
  created_at timestamptz default now()
);

-- עדכונים קוליים מהלקוח
create table voice_updates (
  id uuid primary key default gen_random_uuid(),
  client_id uuid references clients(id),
  audio_url text,
  transcription text,
  summary text,                          -- Claude summary
  created_at timestamptz default now()
);

-- Row Level Security (RLS) - חובה
alter table leads enable row level security;
create policy "Clients can only see their own leads"
  on leads for select
  using (client_id = current_setting('app.client_id')::uuid);
```

### 11.3 מלכודות פסיכולוגיות (שלי, המייסד)

**1. Perfectionism trap**
המוצר לא צריך להיות מושלם. הלקוח הראשון צריך לראות ערך, לא תכנון מושלם.
**אנטי:** משחרר אחרי שבוע גם אם זה 80% מוכן.

**2. Feature creep**
ההזמנה לבנות סוכן חדש לכל בקשה.
**אנטי:** סטיק ל-9 הסוכנים שהוגדרו. כל רעיון חדש נכנס לבק-לוג, לא לבילד.

**3. Stealth mode עד אינסוף**
חשש לצאת לאוויר.
**אנטי:** דדליין קשה — יום 8 = launch ללקוח ראשון, גם אם לא מושלם.

**4. Imposter syndrome**
"אני לא ראוי לגבות ₪900/חודש."
**אנטי:** הלקוח חוסך ₪1,500-2,000/חודש בזמן + מקבל ערך עצום. המחיר מוצדק.
